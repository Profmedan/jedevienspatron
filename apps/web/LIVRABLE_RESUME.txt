   • @/lib/game-engine/types ✓

   • @/components/CarteView ✓

   ─────────

   # Ou en dev: npm run dev

   ───────────────
   Option A (remplacer complètement) :

   // Router selon URL: /jeu → ancien, /jeu?v=2 → nouveau

5. TESTER
   ───────

   # Accéder http://localhost:3000/jeu
